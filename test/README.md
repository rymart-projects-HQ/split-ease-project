# Test Directory

Test configuration and utilities for the Split Ease application.

## Structure

```
test/
└── setup.ts    # Test setup configuration
```

## Setup

The `setup.ts` file contains global test configuration that runs before tests.

## Running Tests

Testing framework to be configured. Recommended options:

- **Vitest** - Fast unit testing with Vite integration
- **Playwright** - End-to-end testing

## Future Testing Plan

- Unit tests for composables and utilities
- Component tests for Vue components
- API route tests for server endpoints
- E2E tests for authentication flows
